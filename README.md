# 🔥 AI Fire & Smoke Detection System

A real-time AI-powered Fire & Smoke Detection System built using YOLOv8, OpenCV, and PyTorch with GPU acceleration.

---

## 🚀 Features

* 🔥 Real-time Fire Detection
* 💨 Smoke Detection
* 🎤 Voice Alert System
* ⚡ GPU Accelerated Inference (RTX Support)
* 🌞 Day & 🌙 Night Detection
* 📷 Live Camera Monitoring
* 🧠 YOLOv8 Custom Trained Model

---

## 🛠️ Tech Stack

* Python
* YOLOv8
* OpenCV
* PyTorch
* CUDA
* NumPy

## ⚡ Installation

### 1️⃣ Clone Repository

```bash
git clone https://github.com/pratiksingh519/AI-Fire-Smoke-Detection.git
cd AI-Fire-Smoke-Detection
```

---

### 2️⃣ Create Virtual Environment

```bash
python -m venv env
```

Activate:

```bash
env\Scripts\activate
```

---

### 3️⃣ Install Dependencies

```bash
pip install ultralytics opencv-python torch torchvision pyttsx3
```

---

## 🚀 Run Detection

```bash
python detect.py
```

---

## 🧠 Model Training

```bash
python train.py
```
